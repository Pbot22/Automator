Machine Locked

Download

Use the download button to get the latest Windows installer. Windows may show a warning for unsigned software.

Install

Run the installer, open Automator, and wait for the activation window to appear.

Send Machine ID

Copy the Machine ID from the activation window and send it to the product owner. The key must be generated for that exact machine.

Activate

Paste the activation key you receive into Automator and click Activate. If the key matches the machine, the app will unlock.
